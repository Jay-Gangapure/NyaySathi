📘 NyaySathi – Real-Time Legal Emergency Assistant

## 🚀 Overview

NyaySathi is a **Real-Time AI Legal Emergency Assistant** designed to help citizens understand their rights and take immediate action in real-life situations.

Instead of complex legal explanations, NyaySathi provides:

* ✅ Instant situation-based guidance
* ✅ Simple, actionable steps
* ✅ Multilingual support
* ✅ Voice-enabled interaction

👉 Think of it as **“Google Maps for legal situations”**

---

## 🎯 Problem Statement

Many citizens:

* Don’t know their legal rights
* Face language barriers
* Cannot afford legal help
* Get exploited due to lack of awareness

NyaySathi solves this by providing **quick, clear, and accessible legal guidance**.

---

## 💡 Solution

A **voice-first, multilingual web platform** where users can:

1. Select a situation (e.g., Traffic Police Issue)
2. Instantly get:

   * Rights
   * What to do
   * What not to do
   * Immediate actions
3. Access help resources and complaint guidance

## ⚙️ Installation & Setup

### Prerequisites

* Node.js (v18+)
* Python (v3.10+)

---

### 🔧 Frontend Setup

```bash
git clone <your-repo-url>
cd mini-project
npm install
npm run dev
```

Open:

```
http://localhost:5173
```

---

### 🔧 Backend Setup

```bash
cd nyaysathi-backend
python -m venv venv
source venv/bin/activate   # (or venv\Scripts\activate on Windows)
pip install -r requirements.txt
uvicorn main:app --reload
```

Open:

```
http://127.0.0.1:8000/docs
```

## 🔄 Project Workflow

1. User selects a situation
2. Frontend sends request to backend
3. Backend fetches structured legal data
4. Response displayed as:

   * Rights
   * Do’s
   * Don’ts
   * Actions
